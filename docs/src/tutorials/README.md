# Tutorials

Step-by-step tutorials for learning Brokkr. Each tutorial walks you through a complete workflow from start to finish, building practical skills along the way.

These tutorials assume you have a working Brokkr development environment. If you haven't set one up yet, follow the [Installation Guide](../getting-started/installation.md) first.

| Tutorial | What You'll Learn |
|----------|-------------------|
| [Deploy Your First Application](./first-deployment.md) | Create a stack, add a deployment object, register an agent, and watch Kubernetes resources get applied |
| [Multi-Cluster Targeting](./multi-cluster-targeting.md) | Use labels and annotations to direct deployments to specific agents |
| [CI/CD with Generators](./cicd-generators.md) | Create a generator and use it from a CI/CD pipeline to push deployments |
| [Standardized Deployments with Templates](./templates.md) | Create reusable templates with JSON Schema validation and instantiate them across stacks |
